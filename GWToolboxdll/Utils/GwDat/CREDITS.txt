The Gw.dat reader and texture decoders in this directory are derived from
GuildWarsMapBrowser by Jonathan Bjørn Greve:

    https://github.com/Jonathan-Greve/GuildWarsMapBrowser

Files derived from that project:
    xentax.cpp / xentax.h          - GWDat file decompression (UnpackGWDat)
    AtexReader.cpp / AtexReader.h   - ATEX/ATTX -> RGBA decoding
    AtexAsm.cpp / AtexAsm.h         - ATEX decompression sub-codes
    AtexDecompress.cpp / AtexDecompress.h - ATEX decompression driver

The MFT parsing and file lookup (in Modules/GwDatModule.cpp) is an
independent reimplementation based on GuildWarsMapBrowser's GWUnpacker.
