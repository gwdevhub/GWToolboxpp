GWToolbox++ — together with the [LFG board](https://lfg.gwtoolbox.com/),
the free [text-to-speech API](https://tts.gwtoolbox.com/), and the
[Kamadan](https://kamadan.gwtoolbox.com/) and
[Ascalon](https://ascalon.gwtoolbox.com/) trade-chat archives — is built and
run by a handful of volunteers in their spare time. There is no company
behind any of it; nobody is paid; nothing is monetised inside the game.

What that means in practice:

* **Servers cost money.** The LFG board, the trade archives, and the TTS
  API all run on small VPSes that someone is paying for out of pocket each
  month. The same goes for the build infrastructure that turns each
  commit into a signed Toolbox release.
* **Time is the bottleneck.** Reverse-engineering the Guild Wars client,
  porting features as new patches land, and answering bug reports happens
  on evenings and weekends. New features land when one of the maintainers
  has a quiet night to write them.

If GWToolbox or any of the related tools has been useful to you and you
would like to chip in, the project is on Open Collective:

<p style="margin: 1.5rem 0;">
  <a href="https://opencollective.com/gwdevhub_collective"
     target="_blank"
     rel="noopener">
    https://opencollective.com/gwdevhub_collective
  </a>
</p>

Open Collective is a transparent funding platform — every contribution is
public and every expense the project pays for (server bills, domain
renewals, code-signing certificates) is itemised on the same page, so you
can see exactly where the money goes.

Donations are entirely voluntary and have **no effect** on what features
land in Toolbox, what gets prioritised in the bug tracker, or how anyone
in the Discord is treated. Reporting a good bug, contributing a fix, or
helping someone in `#help` is just as valuable.

Thanks for using the tools.
